# License Issuer Web UI

The testing web UI is available inside Aptoria at:

```text
/program-settings/license-issuer
```

It is admin-only and wraps the separated issuer logic under `tools/license-issuer/src/LicenseIssuerCore.php`.

This is temporary for testing. Later the entire `tools/license-issuer` folder should be moved to a private repository or private internal utility.
