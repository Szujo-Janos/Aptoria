# Aptoria v1.0.66 – Settings Functional Audit Reality Pass

v1.0.66 was the Settings audit/reality pass. It documented the difference between saved Settings fields and actual runtime behavior, and was superseded by v1.0.67 activation wiring and v1.0.68 test hotfixes.

## Scope

- Reviewed Settings Center controls against runtime consumers.
- Documented where fields were only saved/exported versus where they changed application behavior.
- Preserved Windows/XAMPP and GitHub Actions compatibility.

## Follow-up

The audit presentation was intentionally removed from the product UI after feedback. Current Settings behavior is documented in `docs/SETTINGS_FUNCTIONAL_AUDIT.md`.
